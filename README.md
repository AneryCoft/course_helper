# 课程助手
一个管理学习通、雨课堂课程的应用。

## 功能
- [x] 多账号操作
学习通:
- [x] 签到（所有五种签到任意设置）
- [x] 主题讨论
- [x] 随堂测试（自动提交答案）
- [ ] 作业
- [ ] 群聊签到
- [ ] IM协议
雨课堂: 
- [x] 动态二维码签到
- [x] PPT展示
- [x] 课堂答题
